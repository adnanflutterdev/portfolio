# Setup Instructions

## 1. Clone Repository

```bash
git clone https://github.com/adnanflutterdev/portfolio.git
```

## 2. Open Project

Open the folder in:
- VS Code
- WebStorm
- Any code editor

## 3. Run Website

Simply open:

```bash
index.html
```

Or use VS Code Live Server extension.

---

# Deployment

This website is deployed using Vercel.

## Deploy To Vercel

1. Push code to GitHub
2. Go to https://vercel.com
3. Import GitHub repository
4. Click Deploy
